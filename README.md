PROYECTO JCLOUD

Este es un proyecto personal que se encarga de simular el comportamiento que tiene el servicio s3 de aws. Este servicio es un almcaen de objetos binarios encriptados. Jcloud hace lo mismo, recibe, encripta y almacena.
Lo que me motivo a hacer este proyecto fue la idea de implementar un protocolo personal en la capa de aplicacion. El cliente debe enviar una trama de bytes con un formato especial para que el sistema comprenda la solicitud.
